== Description ==
Swiftea is an open-source tool to power your internal search engine.

By installing this plugin on your site, you use the Swiftea service (https://swifteasearch.alwaysdata.net) which browses and indexes your website.

== Installation ==
Just install & activate the plugin, then use this shortcode where you want to add the search form:
    [swiftea]
